Q. What is the difference between git log and git status?
A. Git log shows a history of all commits and changes to repository whereas git status shows current state of the repository in terms of uncomitted and/or unstaged files.

Q. What command would you use to see the commits where “main.java” was one of the commited files?
A. "git log -- main.java"

Q. What command would you use to see the commits whose commit message contains the word “transaction”?
A. "git log --grep="transaction" "

Q. What command would you use to see the com mits made by “Lauren” between 01/05/2014 and 06/14/2015?
